How to Install and Use Visual Studio Code Live Server Extension

Video install | use

Intallation Instructions

  1. In Studio Code ,click the
  2. In the seach bar that appears, type live server.
  3. Select the live server extensions. It should be the top choice.
  4. In the page for the live server that comes up, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use live server.
  • In VS Code's Explorer, right click(windows) or Control+click (macOS) on a HTML file. From the context menu that appears, select Open with live server.
